# 🍽️ What's For Dinner — Recipe Finder App

A dynamic and interactive **Recipe Finder App** built using **HTML5**, **CSS3**, **Bootstrap 5**, and **JavaScript**.

This project focuses on DOM manipulation, dynamic rendering, and creating interactive user experiences using JavaScript.

---

## 🚀 Live Demo
🔗 https://nadaahmed159.github.io/What-is-For-Dinner/

---

## 📌 Features

✅ Random meal generator  
✅ Dynamic meal rendering using JavaScript  
✅ Interactive tabs for recipe details  
✅ Ingredients, instructions, nutrition & chef tips sections  
✅ Responsive design across all screen sizes  
✅ Smooth hover effects and transitions  
✅ Conditional UI alerts for long preparation meals  
✅ Logic preventing repeated meals consecutively  

---

## 🛠️ Tech Stack

**HTML5** — page structure  
**CSS3** — custom styling and animations  
**Bootstrap 5** — responsive layout and components  
**JavaScript** — dynamic rendering and logic  
**DOM Manipulation** — updating UI dynamically
